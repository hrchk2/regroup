# <Regroup>

## サイト概要
### サイトテーマ
ゲームでのパーティー募集や雑談ができるミュニティサイト

### テーマを選んだ理由
ゲームが好きでいろいろなゲームをしているのですが、一人でやってても飽きてしまったり、攻略するのに人がいたほうが攻略しやすいゲームもあります。そこで、簡単にゲームのパーティー募集ができるサービスがあればと思い、このテーマにしました。

### ターゲットユーザ
ゲーム好きな人

### 主な利用シーン
人数募集時、情報交換
